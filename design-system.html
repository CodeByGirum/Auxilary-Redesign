Philosophy

The Invisible Surface.

Auxiliary is built for professional marketing agencies who manage millions in spend across hundreds of campaigns. The system is designed to handle extreme data density without creating visual noise.

High Velocity

Tabs and lateral drawers allow analysts to switch from campaign auditing to deep SQL analysis in under 200ms.

Client-Ready Reports

AI-generated Notebooks use an editorial layout (Published page) that can be shared with clients without further formatting.

Spatial Awareness

Large rounded corners and gradients help users mentally map their workspace into physical "folders" and "documents."

Advanced Application Guidance

Editorial Emphasis

In Published reports, we utilize a leading-relaxed (1.7) line-height for body text to reduce cognitive load during analysis review.

The 10px Rule

Functional labels (Status, Tags, Metadata) must never exceed 10px but should be paired with font-bold and tracking-widest for legibility.

Numeric Alignment

Always use JetBrains Mono for monetary values in tables. The fixed-width character set ensures columns of numbers align vertically for easy comparison.

The Card Engine.

Our card system reifies abstract campaign data into distinct physical objects. This creates a "workspace" feel rather than a "database" feel.

Projects (Folders)

Corner Radius: 32px standard. Uses an "Inverted Radius" transition for the folder tab to ensure fluid mechanical silhouette.

Gradients: 5-color registry (Blue, Pink, Amber, Emerald, Indigo) mapped to agency client tiers.

Animation: Hover triggers a -translate-y-4 on the 3 internal "Document Illustrations" to simulate opening a physical file.

Notebooks (Docs)

Geometry: 3.8:5 aspect ratio. Optimized for portrait reading preview.

Visual Indicator: Centered document illustration changes icon based on analysis_type (Chart vs Text).

Integration Tiles

Integrations use a 16px radius. Smaller than Folders to signify they are "Utility Components" rather than "Primary Assets."

The Page Registry.

Published Editorial Layout

Inspired by high-end financial news journals. It trades app-like UI for typographic hierarchy to improve long-form reading performance.

Datasets Registry

Uses a 3-column masonry-style layout. Each card is a EmbeddedTable component in its minimized state. Interaction depth: 1 click = Context Menu, 2 clicks = Full Workspace.

Workspace Search

Centralizes project indexing. Features autoFocus on mount and categorized results (Recent vs Trending) to reduce keyboard interaction count.
